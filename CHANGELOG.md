# 版本更新说明

## v0.3.0 - 教材题源练习包

- 新增教材题源表，练习包内容改为优先来自四本本地苏教版教材。
- `db:seed` 会在本地 PDF 存在且 `pdftotext` 可用时抽取教材“练习/习题”写入 SQLite。
- 练习包生成时按学生薄弱知识点匹配教材题源，每个知识点最多取两道教材练习。
- 练习包编辑页展示教材、章节、题源标签和 PDF 页附近信息。
- 教材 PDF 和抽取后的题库仍仅保留在本地，不提交到远端仓库。

## v0.2.0 - 学生中心江苏版

- 移除班级管理与班级诊断，改为教师直接管理学生。
- 系统范围收敛为江苏省高中数学，不再提供广东或通用地区入口。
- 知识点标签改为参考四本苏教版高中数学教材目录：必修第1册、必修第2册、选择性必修1、选择性必修2。
- 错题、诊断和练习包全部以学生为单位流转。
- 工作台改为学生工作台，展示学生、错题、练习包、教材范围和整体薄弱项。
- 教材 PDF 作为本地参考资料保留，不默认提交到远端仓库。

## v0.1.0 - 初始可运行版本

- 搭建 Next.js + React + TypeScript + Prisma + SQLite 项目。
- 实现教师登录、工作台、班级管理、学生档案。
- 实现错题图片上传、本地保存、人工校对与入库。
- 实现知识点、错误类型、江苏/广东/通用地区标签。
- 实现班级与学生诊断看板，包含薄弱知识点、错误类型、近 30 天趋势和到期复习。
- 实现按薄弱知识点生成专项练习包，支持编辑、确认和打印。
- 预留 AI 任务接口；未配置 AI Provider 时自动跳过，不外发学生数据或题图。
- 加入示例教师、班级、学生、知识点、错误类型和错题种子数据。
